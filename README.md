# ai-design-toolkit

Bộ công cụ AI-assisted design cho team MarCom TECOTEC Group.  
Dành cho: Thiện Nghiệp, Thành Đông và các thành viên làm UI/UX.

---

## Bộ công cụ gồm 2 phần

| Tool | Vai trò |
|------|---------|
| [figma-ui-mcp](https://github.com/TranHoaiHung/figma-ui-mcp) | Điều khiển Figma Desktop bằng text command (tạo frame, component, token...) |
| [Impeccable](https://impeccable.style) | Skill + command giúp AI hiểu ngôn ngữ design, tránh lỗi phổ biến |

Hai tool này hoạt động độc lập nhưng kết hợp với nhau cho workflow mạnh nhất.

## Yêu cầu

- macOS (Antigravity hiện chỉ hỗ trợ macOS)
- Figma Desktop (bắt buộc -- figma-ui-mcp không chạy trên Figma web)
- Node.js >= 18
- Tài khoản Antigravity (Google)
